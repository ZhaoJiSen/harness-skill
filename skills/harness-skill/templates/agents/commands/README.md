# Commands

Project-scoped, reusable prompts / slash-commands for AI agents working in this repo.

Drop one Markdown file per command here (e.g. `release.md`, `triage-bug.md`). Each file is the
prompt body an agent runs when the command is invoked. Keep them imperative and specific to this
project's workflow.
